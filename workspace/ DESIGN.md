## Architecture Decision
**Stack:** React Native + Expo Router + Supabase + React Query + TypeScript
**Structure:** Clean Architecture with layers for database, auth, hooks, screens, and components
**Key decisions:**
- Use of Supabase for database and authentication
- Implementation of React Query for state management and caching
- Development of reusable hooks for fetching and manipulating data
- Use of TypeScript for type safety and maintainability
